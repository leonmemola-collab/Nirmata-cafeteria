# Habilidad: Skill Creator

Esta habilidad está diseñada para asistir en la creación de nuevas habilidades, proporcionando una estructura estandarizada y mejores prácticas.

## Archivos Creados

*   `SKILL.md`: Contiene las instrucciones detalladas para el agente.
*   `checklist_skill.md`: Un recurso para verificar la calidad de las habilidades creadas.

## Estructura de la Habilidad

```text
.agent/skills/skill-creator/
├── SKILL.md
└── resources/
    └── checklist_skill.md
```

## Próximos Pasos

Ya puedes usar esta habilidad pidiéndome que cree una nueva habilidad. Por ejemplo: *"Usa la habilidad de creador de habilidades para diseñar una nueva habilidad para analizar logs de error."*

El sistema detectará automáticamente la descripción de la habilidad y cargará las instrucciones cuando sea pertinente.
